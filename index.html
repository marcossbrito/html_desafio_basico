Estrutura básica do HTML

(Voltar)

A estrutura básica do HTML é formada pelas tags html, head, title e body.

Tag <html>

<html> é a tag que vai indicar que essa é uma página escrita em HTML, dentro dessa tag serão adicionadas todas as outras tags necessárias.

Tag <head>

<head> é a primeira tag adicionada dentro da tag html, dentro dela serão adicionadas tags de configuração da página e também referentes ao cabeçalho da página.

Tag <title>

<title> é usada dentro da tag head, ela é usada para o título da página, que será visualizado na aba do navegador.

Tag <body>

<body> é utilizada após a tag <head>; dentro dessa tag serão adicionadas todas as tags para construção do corpo da página, como tags para texto, imagem, entre outras.

Todas essas tags precisam ser abertas e fechadas ao final, para seu correto funcionamento.

Exemplo de abertura e fechamento da tag: <body> conteúdo da tag aqui </body>

Obs.: Nem todas as tags precisam de uma tag de fechamento, por exemplo a tag <input>

Os textos são muito utilizados em HTML, veja alguns exemplos de tags utilizadas para os textos.

Tags de títulos

Tag: <h1>Título 1</h1>
Resultado:

Título 1

Tag: <h2>Título 2</h2>
Resultado:

Título 2

Tag: <h3>Título 3</h3>
Resultado:

Título 3

Tag: <h4>Título 4</h4>
Resultado:

Título 4

Tag: <h5>Título 5</h5>
Resultado:

Título 5

Tag: <h6>Título 6</h6>
Resultado:

Título 6

Outras tags de texto

Parágrafo

Tag de parágrafo: <p>parágrafo 1</p> <p>parágrafo 2</p>
Resultado:

parágrafo 1

parágrafo 2

Sup e Sub

Tag de sup: <p> 10<sup>2</sup> </p
Resultado:

102

Tag de sub: <p> H<sub>2</sub>O </p
Resultado:

H2O

Estilizar o texto

Tag de marcação de texto: <p> Esse texto <mark>possui </mark> marcação</p>
Resultado:

Esse texto possui marcação

Tag sublinhar: <p> Esse texto <u>foi sublinhado</u> </p>
Resultado:

Esse texto foi sublinhado

Tag itálico: <p> Esse texto <i>está em itálico</i> </p>
Resultado:

Esse texto está em itálico

Tag negrito: <p> Esse texto <strong>está em negrito</strong> </p>
Resultado:

Esse texto está em negrito

Tag del: <p> Esse texto <del>foi riscado</del> </p>
Resultado:

Esse texto foi riscado

Tag small: <p> Esse texto <small>é menor</small> </p>
Resultado:

Esse texto é menor

Listas

(Voltar)

Existem dois tipos de listas, ordenadas e não ordenadas, veja os exemplos

Lista Ordenada

Tag:
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>
Resultado:

  1. Item 1
  2. Item 2
  3. Item 3

Lista Não Ordenada

Tag:
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
Resultado:

  • Item 1
  • Item 2
  • Item 3

Links

(Voltar)

Links são necessários para fazer navegação na página e também direcionar para uma página de outro site

Tag: <p> <a href="https://www.w3schools.com/" target="_self">W3 Schools</a> </p>
Resultado:

W3 Schools

Observe que foram usados atributos nessa tag, o primeiro atributo adiciona o link que será aberto, já o segundo atributo define se o link será aberto na mesma aba ou em uma nova aba donavegador, nesse caso será aberta em uma nova aba.
